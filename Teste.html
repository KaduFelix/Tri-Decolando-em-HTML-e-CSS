Meu Aprendizado em HTML e CSS

Cabeçalho

Ao longo do trimestre, aprendi a criar cabeçalhos atraentes utilizando HTML e CSS. Entendi a importância de um bom design para chamar a atenção do usuário e transmitir a identidade visual de um site.

Footer

Também aprendi a criar rodapés eficientes e informativos. Com o uso de HTML e CSS, pude adicionar elementos como links, botões e informações de contato, tornando o rodapé interativo e útil para os visitantes do site.

Variáveis CSS

Descobri a poderosa funcionalidade das variáveis CSS, também conhecidas como custom properties. Com elas, pude definir valores reutilizáveis para propriedades como cores, tamanhos e espaçamentos, facilitando a manutenção e personalização do estilo das páginas.

Responsividade

Aprendi a importância de criar páginas responsivas, que se adaptam a diferentes dispositivos e tamanhos de tela. Utilizando técnicas como media queries, pude tornar meu conteúdo legível e navegável em smartphones, tablets e computadores.

Publicação de Projetos

Ao finalizar meus projetos, aprendi a compartilhá-los com o mundo. Utilizando serviços de hospedagem web, pude publicar minhas páginas para que outras pessoas pudessem acessá-las, avaliá-las e até mesmo contribuir com o código.
