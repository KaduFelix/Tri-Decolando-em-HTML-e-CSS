Meu Aprendizado em HTML e CSS


Bem-vindo(a) à minha página onde compartilho meu aprendizado em HTML e CSS! Durante este trimestre, explorei diversos conceitos e técnicas que me ajudaram a criar páginas web incríveis e responsivas.

Nesta página, você encontrará informações sobre as seguintes áreas do meu aprendizado:

  • Medida REM
  • Media Queries e Responsividade
  • Github e Vercel
  • Flexbox
  • Variáveis em CSS

Continue rolando para obter mais detalhes sobre cada tópico!

REM

Ao longo do trimestre, aprendi a utilizar a unidade de medida REM para definir tamanhos de fonte, espaçamentos e outros elementos de forma mais consistente em relação ao tamanho da raiz do documento.

Media Queries e Responsividade

Também aprendi a utilizar as Media Queries para criar estilos específicos para diferentes tamanhos de tela. Com isso, tornei minhas páginas responsivas, adaptando-as de forma elegante em dispositivos móveis, tablets e desktops.

Github e Vercel

Explorei as plataformas do Github e Vercel para hospedar meus projetos e publicações. No Github, compartilhei meu código e estou ansioso para poder colaborar com outros desenvolvedores. Já no Vercel, pude hospedar minhas aplicações web de forma rápida e segura.

Flexbox

Utilizei o Flexbox para criar layouts flexíveis e responsivos em meus projetos. Com o Flexbox, pude organizar e alinhar os elementos de forma intuitiva, ajustando-os automaticamente conforme o tamanho da tela.

Variáveis em CSS

Outra técnica que explorei foi o uso de variáveis em CSS. Com as variáveis, pude armazenar valores reutilizáveis, como cores, tamanhos e espaçamentos, facilitando a manutenção e a consistência do design em todo o meu código.
